# DocVault

**DocVault** — это легковесное и быстрое Windows-приложение для организации личного архива документов. 

Устали искать сканы паспорта, важные чеки или договоры по бесконечным папкам? DocVault решает эту проблему. Никаких облаков, чужих серверов и платных подписок. Все ваши файлы и база данных хранятся **исключительно на вашем жестком диске**. Полная приватность и мгновенный доступ к нужным бумагам.

---

##  Ключевые возможности

*  **Удобное хранение:** Добавляйте PDF-документы и изображения (JPG, PNG). Приложение само аккуратно скопирует их в свою безопасную локальную папку.
*  **Мгновенный поиск:** Находите нужные документы за доли секунды по названию или пользовательским тегам.
*  **Встроенный предпросмотр:** Изображения отображаются прямо в окне программы, а PDF-файлы безопасно открываются в вашей привычной программе для чтения.
*  **Редактирование на лету:** Ошиблись в названии или решили добавить новый тег? Метаданные можно легко изменить в любой момент.
* 🗑 **Чистое удаление:** При удалении документа программа стирает не только запись из базы данных, но и сам физический файл с диска, не оставляя цифрового мусора.

---

## Установка и запуск

Программа работает полностью автономно. Вам **не нужно** устанавливать среду, скачивать сторонние библиотеки или настраивать базы данных!

1. Перейдите на страницу [Releases](https://github.com/ProstoCheI/DocVault/releases).
2. Скачайте последнюю версию файла `DocVaultLocal.exe`.
3. Поместите файл в любую пустую папку на вашем ПК (Windows 64-bit).
4. Запустите программу!

*(Примечание: При первом запуске DocVault автоматически создаст папку `Storage` для копий ваших файлов и файл базы данных `DocVault.db` рядом с собой).*

## Технологический стек

* **Язык:** C#
* **Платформа:** .NET 9 (Windows Forms)
* **База данных:** SQLite (встроенная)
* **ORM:** Dapper (для быстрых, чистых и безопасных SQL-запросов)
